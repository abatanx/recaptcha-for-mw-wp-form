== Description ==
If MW WP Form plugin and this plug-in are enabled, and it can use reCAPTCHA V3 (https://developers.google.com/recaptcha/docs/v3) in MW WP FORM.

Note:
Required MW WP Form (https://wordpress.org/plugins/mw-wp-form/)

== Installation ==
1. Activate MW WP Form plugin.
2. Upload the entire `recaptcha-v3-for-mw-wp-form` folder to the `/wp-content/plugins/` directory.
3. Activate this plugin through the 'Plugins' menu in WordPress.
